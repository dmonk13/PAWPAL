Here’s a refined, more aesthetic version of your Spotlight tile that keeps all functionality intact (match badge, badges, note input, Like, Woof). It focuses on cleaner hierarchy, softer visuals, and better touch targets.

Design goals

Improve visual hierarchy: photo → name/age → breed/size → distance → health/safety → traits → actions.

Softer, modern card: rounded corners, subtle shadow, airy spacing, consistent iconography.

Accessible and mobile-friendly: larger tap targets, 44px buttons, clear contrast.

Recommended visual changes

Card

Elevation: soft shadow (blur 16, y 6, 10% opacity), radius 16–20.

Padding: 16 all sides; internal grid with 8/12 spacing rhythm.

Divider: 1px subtle line between content and actions.

Photo

Make photo a 5:4 rounded thumbnail with radius 12; edge-to-edge on left, or use a top banner style with 8px inset.

Add a light gradient overlay on the right edge to improve text legibility if text overlaps (if using banner layout).

Header row

Name - age: bold 16–18px.

Match badge: pill with soft gradient, e.g., from #FFE8ED to #FFD1DB; text 12px medium, include a small heart or star icon.

Keep breed/size stacked in 13–14px muted color.

Meta row

Distance: compact location pin icon + “700m away” in 12–13px secondary.

Health badge: pill with soft green background (#E9F8F0) and green text/icon; use an inline shield/check icon.

Traits

Prefix labels in small caps or bold: “Activity:” and “Temperament:”.

Truncate long trait lists after 1–2 lines with “more…” tooltip or expand-on-tap.

Note input

Rounded 12, soft border #E6E8EC, focus ring #8AD1FF.

Placeholder: “Add a note (optional)”.

Character counter subtle, right-aligned 12px secondary.

Auto-grow to 3 lines max; otherwise scroll inside.

Actions

Two equal-width buttons in a safe bottom bar separated by 8px gap.

Like: solid rose/coral; icon left; radius 12; height 44–48.

Woof: outlined warm gold with subtle glow on hover/press; show remaining in a small badge.

Add micro-interactions: tap ripple, 120–160ms scale/opacity.

Micro-animations

On Woof: 300ms pulse of the button outline and confetti shimmer on the card corner.

On Like: small heart pop + haptic light.

Color/typography

Font: Inter/SF/Roboto; headings 600 weight; body 400/500.

Use a cohesive palette:

Primary rose/coral: #FF6B6B

Gold accent (Woof): #F5C04E

Success green: #2BB673 with background #E9F8F0

Text primary: #23262F; secondary: #6B7280; borders: #E5E7EB

JS enhancements (non-breaking)

js
// Character counter
const tiles = document.querySelectorAll('.spotlight-card .note textarea');
tiles.forEach(t => {
  const counter = t.parentElement.querySelector('.counter');
  const max = parseInt(t.getAttribute('maxlength')||'200',10);
  const update = () => counter.textContent = `${t.value.length}/${max}`;
  t.addEventListener('input', update);
  update();
});
